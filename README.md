# 🎯 ExamSphere  
### *(Exam Management System)*

ExamSphere is a desktop-based Online Examination System built using **Java Swing, JDBC, and MySQL**. It provides a complete platform for conducting exams, managing questions, and analyzing results through a clean and modern user interface.

---

## 🚀 Features

### 👨‍💼 Admin Module
- Create and manage exams  
- Add, edit, and delete questions  
- View student results and analytics  
- Leaderboard for performance tracking  

### 👨‍🎓 Student Module
- Register and login securely  
- Attempt timed exams  
- Auto-submit when time ends  
- View results and performance  

---

## ⚙️ Tech Stack

- **Language:** Java  
- **UI:** Java Swing  
- **Database:** MySQL  
- **Connectivity:** JDBC  
- **IDE:** Eclipse  

---

## 🧠 Key Functionalities

- Secure authentication system  
- Timer-based exam system  
- Automatic result evaluation  
- Dynamic question handling  
- Leaderboard ranking system  
- Clean and user-friendly UI  

---

## 🏗️ Architecture

The project follows a **layered architecture**:
- UI Layer (Swing)  
- Business Logic Layer  
- Database Layer (JDBC)  

This ensures better **maintainability, scalability, and structure**.
